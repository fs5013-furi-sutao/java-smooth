## 概要

アルファベットのみで構成される文字列を圧縮する。

## 課題 1. 文字列圧縮

同じの文字が続く場合、"文字+連続する文字数"に置き換え、
文字数を短くして圧縮する StringsCompressor クラスを作成しなさい

```java title=src/StringsCompressor.java（一部抜粋）
string text = encode("AAAAABBBBBBBBBBCDDDDDDDDDEEFFFFFG");
System.out.println(text);
```

```txt title=StringsCompressor.javaの実行結果
A5B10CD9E2F5G
```

## 課題 2. 文字列展開

課題 1 で圧縮した文字列を展開する StringsExpander クラスを作成しなさい

```java title=src/StringsExpander.java（一部抜粋）
string text = decode("A5B10CD9E2F5G");
System.out.println(text);
```

```txt title=StringsExpander.javaの実行結果
AAAAABBBBBBBBBBCDDDDDDDDDEEFFFFFG
```
