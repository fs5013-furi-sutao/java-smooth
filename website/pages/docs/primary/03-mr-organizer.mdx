## 概要

合計金額と、男性、女性の人数から、それぞれの支払金額を計算する。
使用する変数は以下の通りである。

| 変数名         | タイプ | 備考     |
| :------------- | :----- | :------- |
| totalPrice     | 数値   | 合計金額 |
| numberOfMale   | 数値   | 男性の数 |
| numberOfFemale | 数値   | 女性の数 |

## 課題 1

男性、女性の区別はなく、完全な割り勘として計算する。
変数の値が以下の通りであるとき、（出力例）のような表示を行う。

| 変数名         | 値    |
| :------------- | :---- |
| totalPrice     | 12000 |
| numberOfMale   | 3     |
| numberOfFemale | 3     |

```txt title=プログラムの実行結果
男性：2000 円
女性：2000 円
```

## 課題 2

男性は女性よりも 1000 円多く支払う時の計算を行う。

| 変数名         | 値    |
| :------------- | :---- |
| totalPrice     | 12000 |
| numberOfMale   | 3     |
| numberOfFemale | 3     |

```txt title=プログラムの実行結果
男性：2500 円
女性：1500 円
```
