# 35. Java の continue と break

<p class="goal">
  ループ処理で continue や break
  を使って、目的に応じた処理フローを実装できるようになること。
  また、ネストされたループでラベルを使うことができるが、できる限りラベルに頼った処理にしないほうが良いことを知っておく
</p>

## 学習

### continue や break 文を知る

Java の continue 文や break 文とは何かを Web で調べてください。検索キーワードは、「 [java continue break](https://www.google.com/search?q=java+continue+break) 」です。また、continue 文や break 文における「ラベル」の役割についても調べておいてください。

ただし、ラベルを使用すると、処理の流れが読み取りにくくなるため、コードの可読性が低下します。また、Java では一般的にラベルの使用頻度は低いため、ラベルの使用は避けたほうが無難です。
