# 11. IDE の Quick Fix を使う

<p class="goal">
  IDE の Quick
  Fix（修正方法の候補表示機能）を使うと、コンパイルエラーの修正が早くなることを体感してもらう
</p>

## 学習

### VSCode でコンパイルエラーを自動修正する方法を学ぶ

例えば、VSCode で以下のクラスを作って、実行してみてください。

```java title=src/App.java
public class App {
    public static void main(String[] args) {

        Sistem.out.println("Hello world!");
    }
}
```

しかし、残念ながら実行できません。なぜなら、`System` であるべきところが、`Sistem` とつづりが間違っているからです。

VSCode の Quick Fix 機能を使えば、このコンパイルエラーも自分では書き直すことなく、すばやく修正することができます。こういった Quick Fix 機能は、VSCode 以外の他の IDE にも備わっています。

### Quick Fix を行う方法

VSCode ではコンパイルエラー箇所は、赤い波線で強調されます。

![](./images/compile_error.png)

この状態でコンパイルエラー箇所 `Sistem` にマウスカーソルをホバーする（上に乗せる）と以下の 3 点が表示されるはずです。

- 白字でエラー情報（Sistem cannot be resolved クラス「Sistem」を解決できません）
- 青字リンクで Peek Problem
- 青字リンクで Quick fix

![](./images/hover_on_compile_error.png)

この中から `Quick Fix` を押すと、自動修正できる候補のリストが表示されます。

- Create class 'Sistem'
- Create interface 'Sistem'
- Create constant 'Sistem'
- Create enum 'Sistem'
- Create local variable 'Sist
- ** Change to 'System'(java.lang) **
- Create field 'sistem'
- Create parameter 'Sistem'

この候補リストの中から `Change to 'System'(java.lang)` を選びます。

![](./images/quick_fixed_list.png)

そうすると IDE が自動でミスタイピングを修正してくれます（`Sistem` → `System`）。自動修正されただけでは、修正後のファイルは保存されていないので（エディタのタブ右端が白丸だと、変更後未保存状態）、Ctr + S キーを押してファイルを保存します。

![](./images/quick_fixed_java_file.png)

### Quick Fix のショートカットキーは Ctr + .

Quick Fix は前述のように、Quick Fix リンクのクリックでもできますが、ショートカットキーを覚えるとコーディングが速くなります。

Quick Fix をショートカットキーで実施する場合は、文字カーソルをコンパイル部分に置いて、Ctr + .（ピリオド）キーを押します。

## ミニ課題（時間: 20 分）

以下のプログラム QuickFixTester.java を、VSCode の Quick Fix 機能を使って、コンパイルエラーを修正して、プログラムを実行できるようにしなさい

ただし、修正するにあたって Quick Fix 機能のみを使い、文字の削除・入力でコンパイルエラー部分を修正しないこと

```java title=src/QuickFixTester.java
public class QuickFixTester {
    public static void main(String[] args) {

        rand = new Random();
        myName = "山本一郎";
        boolean isLive = rand.nextBoolean();

        snowMessage(myName, isLive);
    }

    private static void showMessage(String name, boolean isLive) {
        if (isLive) {
            System.out.println("%s は生きてるよ.", name);
        } else {
            System.out.println("%s は死んでるよ.", name);
        }
    }
}
```
