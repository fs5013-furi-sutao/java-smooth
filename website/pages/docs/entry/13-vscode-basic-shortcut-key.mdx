# 13. ⌨ 1. 基本的なショートカットキー

## 学習

### VSCode の基本的なショートカットキー

VSCode では、作業を効率化できるショートカットが多数用意されています。ここでは、VSCode でよく使う基本的なショートカットキーを挙げておきます。どのツールでもいえることですが、ショートカットキーを覚えることで作業効率がアップします。

## 基本操作

まずは基本操作に関するショートカットをご紹介します。コーディングをしていると、タブの移動、ファイルの移動などが頻繁に行われます。基本操作のショートカットを覚えるだけでも、作業がかなりはかどりますよ。

| 操作                   | Windows          | Mac             |
| :--------------------- | :--------------- | :-------------- |
| ファイルを開く         | Ctrl + O         | Cmd + O         |
| 新しいファイルを開く   | Ctrl + N         | Cmd + N         |
| エクスプローラーを開く | Ctrl + Shift + E | Shift + Cmd + E |

### エクスプローラー

エクスプローラーとは、画面左にあるアクティビティバーの一番上のアイコンを開くとサイドバーに表示される項目で、現在開いているフォルダーや、ファイルを確認することができます。

![エクスプローラー](./images/vscode_explorer.png)

<br />

| 操作                       | Windows          | Mac             |
| :------------------------- | :--------------- | :-------------- |
| ファイルの保存             | Ctrl + S         | Cmd + S         |
| 名前を付けてファイルを保存 | Ctrl + Shift + S | Shift + Cmd + S |
| クイックオープンウィンドウ | Ctrl + P         | Cmd + P         |

### クイックオープンウィンドウ

クイックオープンウィンドウは、画面上部に開く入力窓でファイルを開いたり、指定した行にジャンプしたりすることができます。ここの窓に `>` を入力すると、コマンドパレットのように VSCode の機能を検索することができます。ここに `?` を入力すると使い方が表示されます。

![クイックオープンウィンドウ](./images/vscode_quick_open_window.png)

<br />

| 操作                   | Windows          | Mac             |
| :--------------------- | :--------------- | :-------------- |
| コマンドパレットを開く | Ctrl + Shift + P | Cmd + Shift + P |

### コマンドパレット

クイックオープンウィンドウに `>` が表示された状態で入力窓が開きます。機能の検索やショートカットキーの確認をする場合は、こちらで開いて検索した方が早いです。

| 操作                  | Windows    | Mac      |
| :-------------------- | :--------- | :------- |
| サイドバーの開閉      | Ctrl + B   | Cmd + B  |
| エディターの分割      | Ctrl + \   | Cmd + \  |
| タブの移動 Ctrl + tab | Ctrl + tab |

切り替えられるタブの候補が上部に表示されますので、矢印キーで選択します。

![タブの切り替え](./images/vscode_tab_changer.png)

<br />

## コーディング

コーディングによく使うショートカットです。選択行の移動や削除、コピーなどは、地味によく使うショートカットです。カーソルの移動や文字の置き換えは別の項目をご覧ください。

| 操作           | Windows                | Mac                       |
| :------------- | :--------------------- | :------------------------ |
| 選択行の移動   | Alt + 矢印キー         | Option + 矢印キー         |
| 選択行の削除   | Ctrl + Shift + K       | Cmd + Shift + K           |
| 選択行のコピー | Alt + Shift + 矢印キー | Option + Shift + 矢印キー |

### 行の移動・削除・コピー

行の移動、削除やコピーは、カーソルが置いてある行で判断しますので、行の全てを選択しなくても大丈夫です。ザクっと選択してキー操作してください。

| 操作                   | Windows              | Mac                 |
| :--------------------- | :------------------- | :------------------ |
| カーソルの下に行を挿入 | Ctrl + Enter         | Cmd + Enter         |
| カーソルの上に行を挿入 | Ctrl + Shift + Enter | Shift + Cmd + Enter |
| 選択行をコメント化     | Ctrl + /             | Cmd + /             |

## 文字の置換・検索

文字の置換や検索は、作業しているファイルだけでなくワークスペース全体でも行うことが出来ます。用途に応じて使い分けましょう。

| 操作       | Windows  | Mac              |
| :--------- | :------- | :--------------- |
| 文字の置換 | Ctrl + H | Option + Cmd + F |
| 文字の検索 | Ctrl + F | Cmd + F          |

文字の置き換えや検索は、キー操作を行ってから文字列を入力して実行する方法と先に文字列を選択してから実行する方法があります。文字列を先に選択してから実行すると、入力窓に文字列が表示された状態で実行する事が出来ます。

| 操作                     | Windows          | Mac             |
| :----------------------- | :--------------- | :-------------- |
| ワークスペース全体を置換 | Ctrl + Shift + H | Shift + Cmd + H |
| ワークスペース全体を検索 | Ctrl + Shift + F | Shift + Cmd + F |

## 指定の場所へカーソルを移動

キーボードから手を離して、マウスに持ち替えカーソルを移動・・・。すごく面倒ですよね。ちょっとした事でも、数が多くなると結構なタイムロスです。ここもショートカットを覚えて、効率化しましょう！

| 操作                     | Windows           | Mac              |
| :----------------------- | :---------------- | :--------------- |
| 文字の先頭へジャンプ     | Home              | Home             |
| 文字の最後へジャンプ     | End               | End              |
| 行を指定してジャンプ     | Ctrl + G          | Cmd + G          |
| ファイルの先頭へジャンプ | Ctrl + Home       | Cmd + Home       |
| ファイルの最後へジャンプ | Ctrl + End        | Cmd + End        |
| 対となる括弧にジャンプ   | Ctrl + Shift + \  | Shift + Cmd + \  |

## 選択（複数選択・矩形選択）

VSCode では選択箇所を同時に編集できる、マルチカーソルの機能が使えます。マルチカーソルを使う上で、選択系のショートカットは覚えておくととても役に立ちますよ。

| 操作                 | Windows                       | Mac                                |
| :------------------- | :---------------------------- | :--------------------------------- |
| 任意の場所を複数選択 | Alt + Click                   | Option + click                     |
| 同じ文字列を複数選択 | Ctrl + D                      | Cmd + D                            |
| 矩形選択             | Ctrl + Shift + Alt + 矢印キー | Shift ＋ Option ＋ Cmd ＋ 矢印キー |

矩形選択（くけいせんたく）とは長方形に選択する事です。通常の選択では、別の行に入ると開始位置から選択されますが、矩形選択では、別の行に入っても行の途中から選択する事ができます。
