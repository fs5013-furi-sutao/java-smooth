# 14. ⌨ 2. ショートカットキーリスト

## 一般

| コマンド                        | Windows          | macOS   |
| :------------------------------ | :--------------- | :------ |
| すべてのコマンドの表示          | Ctrl+Shift+P, F1 | ⇧⌘P, F1 |
| クイックオープン                | Ctrl+P           | ⌘P      |
| 新しいウィンドウ                | Ctrl+Shift+N     | ⇧⌘N     |
| ウィンドウを閉じる              | Ctrl+Shift+W     | ⌘W      |
| ユーザー設定を開く              | Ctrl+,           | ⌘,      |
