# 33. Java の拡張 for 文

<p class="goal">
  拡張 for 文を使うことで、より簡潔に、直感的にループ処理を実装できることを知る
</p>

## 学習

### Java の便利な 拡張 for 文を知る

Java の拡張 for 文とは何か、拡張 for 文を使う方法を Web で調べてください。検索キーワードは、「 [java 拡張 for](https://www.google.com/search?q=java+拡張+for) 」です。3 種類ぐらい説明記事やサンプルコードを読んで、for 文とは何かを知り、for 文の使い方をつかみましょう。

拡張 for 文は、for 文で対象の要素を 1 つずつ取り出す動きをするので「for-each 文」（1 つずつ for 文を回す、の意）とも呼ばれます。

拡張 for 文は、Java のバージョン「Java SE 5.0」で追加された機能です。現状で Java SE 5.0 のようなレガシーなバージョンを使用しているアプリやシステムはごくわずかだと考えられますが、Java SE 5.0 より前のバージョンでは、拡張 for 文が使えないので注意が必要です。

## ミニ課題 01: 拡張 for 文で配列を使う（時間: 20 分）

以下の要素を持った配列「numbers」のすべて要素を合計して、結果を表示する処理を main メソッドに持つ ArrayElementsAdder クラスを作成しなさい

### 配列の要素

- 3, 4, 5, -5, 0, 12

### ArrayElementsAdder の実行結果

```
配列 numbers の 6 個の要素の合計 => 19
```
