# 05. Java 基本型のキャスト

<p class="goal">
  アップキャスト（ワイドキャスト）、ダウンキャスト（ナローキャスト）の違いを知る、キャストができるようになること

## 学習

### キャストを知る

キャストとは何かを Web で調べてください。検索キーワードは、「 [java 基本型 キャスト](https://www.google.com/search?q=java+基本型+キャスト) 」です。

※ここでは参照型のキャストは、現在の学習範囲外なので気を付けてください。基本型のキャストのみを学習してください

## ミニ課題 1: キャスト（時間: 20 分）

main メソッドに以下の 5 つの処理を記述して、プログラム実行結果が以下に示したようになるように Casting クラスを作成しなさい。

### 1. 変数宣言、変数への代入処理:

- double 型の変数「myDouble」に 1.9 を代入

### 2. ダウンキャスト処理:

- 変数 `myDouble` を int 型の `downToInt` にキャスト

### 3. 表示処理:

- 変数 `myDouble` と `downToInt` の内容を表示

### 4. アップキャスト処理:

- 変数 `downToInt` を float 型の `upToFloat` にキャスト

### 5. 表示処理:

- 変数 `downToInt` と `upToFloat` の内容を表示

### プログラム実行結果

```
ダウンキャスト double -> int
1.9 -> 1

アップキャスト int -> float
1 -> 1.0
```

## ミニ課題 2: 自動キャスト（時間: 20 分）

main メソッドに以下の 4 つの処理を記述して、プログラム実行結果が以下に示したようになるように AutoUpcasting クラスを作成しなさい。

### 1. 変数宣言、変数への代入処理:

- char 型の変数「myChar」に a を代入

### 2. 自動キャスト処理:
- int 型の変数「ascii」に `myChar` を代入

### 3. 表示処理:

- 変数 `myChar` と `ascii` の内容を表示

### 4. アップキャスト処理:

- 変数 `myChar` を int 型の `castAscii` にキャスト

### 5. 表示処理:

- 変数 `myChar` と `castAscii` の内容を表示

### プログラム実行結果

```
char -> int への自動キャスト
a -> 97

char -> int へのアップキャスト
a -> 97
```
