# 32. Java の for 文

<p class="goal">
  for
  文を使ってループ処理を実装できるようになること。特定回数ループさせたり、配列の処理に
  for 文のループの相性が良いことを実感していること
</p>

## 学習

### Java の for 文を知る

プログラミングにおける for 文、for ループとは何か、Java で for 文を使う方法を Web で調べてください。検索キーワードは、「 [java for](https://www.google.com/search?q=java+for) 」です。3 種類ぐらい説明記事やサンプルコードを読んで、for 文とは何かを知り、for 文の使い方をつかみましょう。
