# 16. 文字列入力

  ユーザからの数値・文字列の入力をともなうコンソールプログラムを作成できるようになること

## 学習

### 文字列操作

Java のコンソールアプリケーションへのキーボードからの入力は、「標準入力」と呼ばれます。

Java で標準入力を受け付けるには 2 つの方法があります。1 つが BufferedReader クラスを使った方法で、もう 1 つが Scanner クラスを使った方法です。

BufferedReader と Scanner の違いは、検索キーワードは、「 [java bufferedreader scanner 違い](https://www.google.com/search?q=java+bufferedreader+scanner+違い) 」で調べましょう。

現在、一般的には、処理をより簡潔に書ける Scanner クラスで入力処理を実装します。

Scanner クラスで標準入力を受け付ける方法は、Web で調べてください。検索キーワードは、「 [java scanner 入力](https://www.google.com/search?q=java+scanner+入力) 」です。3 種類ぐらい説明記事やサンプルコードを読んで、Java における文字列入力の概要を調べてみましょう。
