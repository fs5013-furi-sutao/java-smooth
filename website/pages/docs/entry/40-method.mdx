### main メソッドからのメソッドの呼び出し方について

main メソッドからメソッドを呼び出す方法には、次の 2 つの方法があります。

1. Static メソッドを呼び出す方法
2. クラスをインスタンス化して呼び出す方法

#### 1. Static メソッドを呼び出す方法

メソッドを main メソッドにそのまま書くだけで、メソッドを呼び出すことができます。ただし、メソッドには修飾子として `static` キーワードを付ける必要があります。なぜなら、static メソッドから、そのまま呼べるメソッドは static メソッドだけだからです。

main メソッドは static メソッドなので、メソッドをそのまま書いて呼び出すには、メソッドを static にしておく必要があります。static メソッドは、クラス固有のメソッドとして扱われるので「クラスメソッド」とも呼ばれます。

以下の `printMessage("Hello!");` がメソッド呼び出しです。

```java title=src/App.java
public class App {
    public static void main(String[] args) {

        printMessage("Hello!");
    }

    public static void printMessage(String message) {
        System.out.println(message);
    }
}
```

#### 2. クラスをインスタンス化して呼び出す方法

メソッドに `static` キーワードを付けないと、そのメソッドは「インスタンスメソッド」として取り扱われます。

インスタンスとは、クラスを実体化したものです。

クラスは設計図で、インスタンスはその設計図に従って実体化したものといわれます。

また、別の例えでは、よく「タイ焼きの型」と「タイ焼き」の関係にも例えられます。1 つのタイ焼きの型からは、複数のタイ焼きが作られるように・・・1 つのクラスからは、何個もインスタンスを作ることができます。

クラスからインスタンスを作る場合には new 演算子を使います。以下のコードの `new App();` がインスタンスの生成にあたります。

インスタンスメソッドを利用するには、このインスタンスからメソッドを呼び出す必要があります。例えばインスタンス `app` からメソッドを呼び出すには、ドット演算子 `.` を使って、`app.printMessage("Hello!");` とメソッドを呼び出します。

static メソッドの中から、`printMessage("Compile error!");` とそのままインスタンスメソッドを呼び出そうとすると、コンパイルエラーになります。

```java title=src/App.java
public class App {
    public static void main(String[] args) {

        App app = new App();

        app.printMessage("Hello!");

        // メソッドをそのまま呼び出すとコンパイルエラーになる
        // printMessage("Compile error!");
    }

    public void printMessage(String message) {
        System.out.println(message);
    }
}
```
