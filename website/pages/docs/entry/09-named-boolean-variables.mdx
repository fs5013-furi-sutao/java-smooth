# 09. 真偽値の変数名の付け方

<p class="goal">
  is, has, can
  などから始まる変数名を付けることで、中身が真偽値（boolean）であることが容易に分かり、どういった場合に
  true になるか、false になるかが、分かりやすくなることを知る
</p>

## 学習

### boolean 型の変数名は is 〇〇 という形

Java に限らず、プログラミングでは真偽値（boolean）型の変数名に、`is〇〇` という形を用いることが、開発者の中での通例となっています。

なぜ、`is〇〇` という形が良いのでしょうか？次の例で考えていきます。

### 性別で考えてみる

例えば、性別をあらわす変数を考えてみます。変数を次のように宣言します。

```java
boolean sex;
```

では、sex が男性を示す場合は「男性です」、女性を示す場合は「女性です」と表示するプログラムを作ってみます。

```java
if (sex) {
    System.out.println(A);  // 男？女？
} else {
    System.out.println(B);  // 男？女？
}
```

さて、A と B のどちらに `"男性です"` と入れれば良いでしょうか？

`sex = true` の時が男性なのか、`sex = false` の時が男性なのか、このプログラムからではわかりません。
プログラムの仕様や設計を確認する必要があります。

しかし、`boolean isMale;` と宣言するとどうでしょうか？
先ほどと同じように、isMale が男性を示す場合は"男性です。"、女性を示す場合は"女性です。"と表示するプログラムを作ってみます。

```java
if (isMale) {
    System.out.println(A);  // 男
} else {
    System.out.println(B);  // 女
}
```

`isMale` を日本語にすると「男性ですか？」という意味になります。

つまり、`isMale = true` であれば、男性を指していることになります。逆に `isMale = false` なら、男性ではないので女性となります。

ですから、A に「男性です。」、B に 「女性です。」 と記述すれば良いわけです。

このように、`is〇〇` という変数名を使うと、ture の時に何を意味し。false の時に何を意味するかを、変数名を見ただけで理解できるようになります。
こういった利点があるので、`is〇〇`といった変数名がプログラミングでは一般的になっています。

変数やメソッドの意味合いによっては、`has〇〇` を使う場合もあります。

例えば `Iterator.hasNext()` は、「次の要素を持っているか？」という意味のメソッドです。`hasNext() == true` なら次の要素があり、`false` なら次の要素がないと分かります。このことは `hasNext()` の仕様を確認しなくても、メソッド名から推測できます。

動作が推測しやすいということは、あなた以外の他の開発者にとっても読みやすいコードとなり、メンテナンスのしやすいコードにつながります。

## ダメなメソッド名の例

よくダメなコードで見るのが、judge なんとかというメソッド名です。

このメソッド名を見た瞬間に、そのソースコード全体のクオリティは低いだろうなと想像してしまいます。

たぶん「判断」のつもりで judge としていると思うんですが、プログラムは常に何かの判断があるのでメソッド名が処理の説明になっていません。

メソッド名は、パッと見てなんとなく処理が分かるようにつけなきゃいけない。そうでないと不具合調査や改修の際に、後々、膨大な工数として跳ね返ってくるという意識が必要です。

### 「電話番号が設定されているか判断する」というメソッド

たとえば「電話番号が設定されているか判断する」というメソッドに対して、次のようなメソッド名は処理の説明になっていないです。

```java
boolean judgePhoneNumberSetting()
```

PhoneNumber の Setting を judge するって、設定されているかどうかなのか、それとも特定の番号かチェックしているのか、中身を見るまで分からないです。

このような感じのコードであると、処理を読む手間がどんどん掛かってしまいます。このメソッド名を改善（リファクタリング）するなら次のような感じです。

```java
boolean isPhoneNumberSet()
```

シンプルで、処理しようとしていることの意図が、すぐに伝わります。
