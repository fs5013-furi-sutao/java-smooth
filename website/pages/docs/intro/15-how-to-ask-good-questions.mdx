# 15. スキルアップできる質問の方法

<p class="goal">
  「どうやって」「どういうふうに」質問するかで、飛躍的にスキルアップできるか、できないかが決まることを知る
</p>

** この項目には、ミニ課題はありません **

## 学習

### 質問内容をまとめるのに時間を掛けるのはもったいない

ひどい言い方かもしれませんが・・・
まとめるのに時間を使うのは、あほらしいので、
「質問したいので折り返し電話ください」でもいいですよ

都合悪い時間あれば、伝えておいてください

### 質問のしかたで、スキルアップできるか、まったくスキルアップできないかが決まる

できるなら、質問する時は

「こうだと思っているんですけど」「こうやるつもりですけど」
という認識合わせで、橋本が OK / NG を
判定するだけの状態にできていると

飛躍的にスキルアップすると思います
（無理して調査に時間を掛けるぐらいなら、
まったく分かりません、で聞いたほうが良いですが）
