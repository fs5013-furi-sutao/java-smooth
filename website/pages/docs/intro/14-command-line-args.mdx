# 14. コマンドライン引数

<p class="goal">
  コマンドライン引数とは何かが分かり、コマンドライン引数を渡して処理を実施できるようになること
</p>

## 学習

コマンドライン引数とは何かということと、コマンドライン引数を渡して Java のプログラムを実行する方法を Web で調べてください。

検索キーワードは、「 [java コマンドライン引数](https://www.google.com/search?q=java+コマンドライン引数) 」です。3 種類ぐらい説明記事やサンプルコードを読んで、コマンドライン引数とは何か、コマンドライン引数を渡して Java のプログラムを実行する方法を習得しましょう。

## ミニ課題（時間: 20 分）

以下に示すプログラム実行結果となる main メソッドを持つ CommandLineArgsTester クラスを作成しなさい。また、コマンドライン引数として「12 山本一郎 670」を渡して、プログラムを実行しなさい

### プログラム実行結果

```
ぼくは 12 歳の 山本一郎 です. 体重は 670 キロです.
```
