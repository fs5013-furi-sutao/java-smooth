## 学習

### Java の開発に必要な 2 点

Java の開発環境には、以下の 2 つを揃える必要があります。

- JDK
- IDE（統合開発環境）

### Java で利用される IDE（統合開発環境）の種類

IDE（統合開発環境）は、コーディングをするエディタのことです。Java の開発ででよく使われるエディタには、以下のものがあります。

- Eclipse
- VSCode
- NetBeans
- IntelliJ IDEA

この中で、IntelliJ IDEA だけは有償の IDE です。現状、ユーザが最も多いと思われる IDE は Eclipse です。

今回のこの学習カリキュラムでは、上記の IDE のうち、VSCode を利用します。

### 今回の環境構築方法

最も一般的な Java 開発環境の構築方法は、各インストーラ（exe ファイル）や zip ファイルをダウンロードしてきて、画面をポチポチしながら環境を作る方法です。

今回は、そのような環境構築方法は行いません。

環境の作成、削除がしやすい、アプリ開発の環境構築にメリットの多い「パッケージ管理ツール」を利用した方法を、今回は採用します。

### パッケージ管理ツール

今回はパッケージ管理ツールとして、Scoop を利用します。

ただし、Scoop は Windows 10 向けのツールなので、開発 PC が Mac OS の場合は、パッケージ管理ツールに HomeBrew を利用してください。

### 今回の環境構築手順

今回の環境構築は、ざっくりと以下の手順で行います。

1. Scoop のインストール
2. Git のインストール
3. OpenJDK 11 のインストール
4. ポータブル版 VSCode のインストール
